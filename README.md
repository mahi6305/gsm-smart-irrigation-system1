# GSM Based Smart Irrigation System

## Introduction
This project monitors soil moisture and controls water pump automatically.
It sends SMS alerts using GSM module.

##  Components Used
- Arduino UNO
- GSM Module (SIM800L)
- Soil Moisture Sensor
- Relay Module
- Water Pump

## Working
1. Soil moisture sensor detects water level.
2. If soil is dry → Pump ON.
3. If soil is wet → Pump OFF.
4. SMS alert sent to farmer.
